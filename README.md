# Diabetes-Prediction-Using-Decision-Tree
A machine learning project that predicts diabetes using a Decision Tree classifier on the PIMA Indians Diabetes dataset.
This project predicts whether a person has diabetes using a Decision Tree classifier.
The PIMA Indians Diabetes dataset is used for training and evaluation.

## Dataset
- PIMA Indians Diabetes Dataset

## Steps Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Decision Tree Model Training
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
